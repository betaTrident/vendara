# Design Setup

Use this command to add the Vercel design system scaffold referenced for the project:

```bash
npx getdesign@latest add vercel
```

## UI Direction

- Primary component library: `shadcn/ui`
- Styling system: `Tailwind CSS`
- Design emphasis: lightweight, mobile-first, practical business UI
- Main UX surfaces:
  - product pricelist management
  - product price history
  - customer running ledger
  - debt and payment timeline
