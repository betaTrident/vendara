# Enterprise Controls

This file captures governance expectations for this repository.

## Baseline

- Keep install manifests, audit allowlists, and agent baselines under review.
- Treat generated workflow files as reviewed artifacts before broad rollout.

## Approval Expectations

- Security-sensitive workflow changes require explicit reviewer acknowledgement.
- Audit suppressions must include a reason and the narrowest viable matcher.
- Generated skills should be reviewed before broad rollout to teams.
